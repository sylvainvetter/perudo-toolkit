# M1 — Probability Calculator

## Overview

M1 computes the probability that a Perudo bid `(q, v)` is satisfied given:
- the player's own visible dice
- the number of hidden opponent dice

The model: total count `T = own_count + X` where `X ~ Binomial(n_unknown, p_die)`.

## Per-die probability

| Condition | p_die |
|---|---|
| Normal play, value 2..6 | 2/6 (die shows v, or shows 1 as joker) |
| Normal play, value 1 | 1/6 (only literal 1s count) |
| Percolateur, any value | 1/6 (no jokers) |

## API

### High-level

```python
from perudo.core.types import Bid
from perudo.m1 import bid_stats

stats = bid_stats(
    bid=Bid(quantity=4, value=3, player_id=0),
    own_dice=[3, 3, 1, 5, 2],   # 2 threes + 1 joker = own_count 3
    n_unknown=15,
    percolateur=False,
)
print(f"P(bid true)  = {stats.p_true:.1%}")   # P(T >= 4)
print(f"P(bid exact) = {stats.p_exact:.1%}")  # P(T == 4)
print(f"Expected T   = {stats.expected:.1f}") # E[T]
```

### Low-level functions

```python
from perudo.m1 import p_at_least, p_exactly, expected_count, distribution

# P(T >= q)
p_at_least(q=4, n=15, p=2/6, own_count=3)

# P(T == q)
p_exactly(q=4, n=15, p=2/6, own_count=3)

# E[T]
expected_count(n=15, p=2/6, own_count=3)

# Full PMF array: dist[k] = P(T == k)
dist = distribution(n=15, p=2/6, own_count=3)
```

## Invariants

| Invariant | Formula |
|---|---|
| Normalisation | `sum(distribution(n, p, own)) ≈ 1.0` |
| Complement | `P(T >= q) + P(T < q) ≈ 1` |
| Monotonicity | `P(T >= q) >= P(T >= q+1)` |
| Deterministic (n=0) | `distribution(0, p, k)[k] == 1.0` |
| Own count satisfied | `own_count >= q → P(T >= q) == 1.0` |
